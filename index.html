ℹ️ Informations clés

Protocole : HTTPS uniquement
Format : JSON (application/json)
Auth : Basic Auth (login:password en Base64)
Host cloud : api.akuiteo.fr
Date format : RFC 3339 (yyyy-MM-dd'T'HH:mm:ss.SSSXXX)
Version : API v2.0.0 (Akuiteo v5)
